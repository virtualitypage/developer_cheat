══❚█ MacOS や Windows で使用可能な Python スクリプト █❚══

chmod755.py

Usage: 指定したディレクトリ内の全ファイルにアクセス権限(読み取り・書き込み・実行)を付与するPythonファイル ※/downloadに"chmod"というフォルダを作成、その中に権限付与したいファイルを入れる

csv_to_json.py

Usage: カレントディレクトリ内の全csvファイルをjsonファイルに変換するPythonファイル ※csvファイルの1行目にはjsonのキー値を入れること

csv_to_txt.py

Usage: カレントディレクトリ内の全csvファイルをtxtファイルに変換するPythonファイル ※csvファイルの1行目にはjsonのキー値を入れないこと

folder_path.py

Usage: 指定したディレクトリ内のフォルダ構成をファイルに出力するPythonファイル(MacOS環境用) ※/User/XXXXに"フォルダ"というフォルダを配置することで使用可能

folder_path_win10.py

Usage: 指定したディレクトリ内のフォルダ構成をファイルに出力するPythonファイル(Windows環境用) ※/User/XXXXに"フォルダ"というフォルダを配置することで使用可能

miruden_chart_csv_to_json.py

Usage: "電気料金表.csv"を"電気料金データ.json"に変換するPythonファイル

miruden_chart_json_to_csv.py

Usage: "電気料金データ.json"を"電気料金表.csv"に変換するPythonファイル

mv_volumes.py

Usage: 外部ディスクのデータを別のディスクにコピーするPythonファイル(MacOS環境用)

mv_volumes_win10.py

Usage: 外部ディスクのデータを別のディスクにコピーするPythonファイル(Windows環境用)

output_csv.py

Usage: テキストの内容をcsvに出力するPythonファイル(MacOS環境用)(XXX_template.shで使用するcsvを作成する) ※/User/XXXX/Desktop/input.txt が読み込み対象

output_csv_win10.py

Usage: テキストの内容をcsvに出力するPythonファイル(Windows環境用)(XXX_template.shで使用するcsvを作成する) ※/User/XXXX/input.txt が読み込み対象

pdf_read.py

Usage: PDFに記載されているテキストをファイルに出力するPythonファイル(MacOS環境用) ※/Desktopに"PDF.pdf"というファイルを配置することで使用可能

pdf_read_win10.py

Usage: PDFに記載されているテキストをファイルに出力するPythonファイル(Windows環境用) ※/User/XXXX/に"PDF.pdf"というファイルを配置することで使用可能

submitter.py

Usage: GUI上のプルダウンメニューを操作して選択した内容をファイルに出力する汎用Pythonファイル(MacOS、Windows両対応) ※ファイルは/User/XXXX/に保存される

text_to_json-title_url.py

Usage: カレントディレクトリ内の全テキストファイルをjsonとcsvファイルに変換するPythonファイル ※キー値は"TEXT"と"URL"の2つが指定されている

users_path.py

Usage: Usersディレクトリ内のフォルダ構成をファイルに出力するPythonファイル(MacOS環境用)

users_path_win10.py

Usage: Usersディレクトリ内のフォルダ構成をファイルに出力するPythonファイル(Windows環境用)
